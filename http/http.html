# http

TCP/IP是互联网相关的各类协议族的总称

TCP/IP协议族里最重要的一点是分层，TCP/IP协议族按层次分别分为以下四层：应用层、传输层、网络层、数据链路层

# 应用层

应用层决定了向用户提供应用服务时通信的活动。 TCP/IP 协议族内预存了各类通用的应用服务。
